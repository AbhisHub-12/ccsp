# Networking Basics 🌐

A standalone, deep networking reference for the CSPP program — the foundation
every other course builds on. Use it as a question-by-question handbook while
studying for labs, interviews, and exams.

## Contents
- 🌐 [Networking Basics — Deep Reference](networking-basics.md)

Covers: OSI & TCP/IP models · TCP vs UDP & the three-way handshake ·
DNS / PING / ARP · network performance metrics · routing, ports, RTT, cast
types & MAC · DHCP + the DORA process · IPv4 vs IPv6 · IP address classes ·
APIPA, switching, hubs, error types & CRC · network topologies · VLAN & VPN ·
firewall, access & trunk ports · NAT/PAT · Telnet vs SSH.

> Definitions are preserved verbatim from the source study material; Mermaid
> diagrams, comparison tables, and callouts are added in the repo's house style.
